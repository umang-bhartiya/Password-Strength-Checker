# Password Strength Checker

A Python-based GUI application that evaluates password strength, generates secure passwords,
and provides real-time feedback with visual indicators.

## Features
- Password strength evaluation
- Strength meter and heatmap
- Secure password generator
- Clipboard support
- Dark mode
- Have I Been Pwned breach check
- Modular architecture (GUI + logic separation)

## Installation
```bash
pip install -r requirements.txt
python gui_app.py
